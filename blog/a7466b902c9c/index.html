说明：运行环境为虚拟机安装的centOs7操作系统（下载地址），安装的docker版本为19.03.12。环境准备阶段不做赘述，下面从安装jenkins开始：

  1. jenkins准备阶段

    • 新建一个目录jenkins_home，用来挂载到jenkins容器里面，可以保存配置
      mkdir -p /var/workspace-20200824/jenkins_home
    • 拉取jenkins镜像
      docker pull jenkins/jenkins:lts
    • 启动jenkins 
      docker run -u root --name jenkins_node \
-p 9080:8080 -p 50000:50000 \
-v $(which docker):/usr/bin/docker \
-v /var/run/docker.sock:/var/run/docker.sock \
-v /var/workspace-20200824/jenkins_home:/var/jenkins_home \
-e TZ="Asia/Shanghai" -v /etc/localtime:/etc/localtime:ro \
jenkins/jenkins:lts
